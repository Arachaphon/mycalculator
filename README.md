**MyCalculator App| React + Vite**

A calculator application built using React and Vite. Designed for clean user interaction and and Tailwind Css for modern styling.

Live Demo:https://mycalculator-six-delta.vercel.app/

UI/UX Design:https://www.figma.com/design/jqWo1T8asW8mdPGDdVRK0O/Untitled?node-id=0-1&t=V9bXkPTGfDI57qAq-1

---

#Key Features

* **Responsive Design:** Ensures seamless functionality and layout across all devices using **Tailwind's Utility Classes**.
* **Intuitive Calculation Logic:** Supports standard arithmetic operations (+, -, x, /) with reliable result handling.
* **Modern State Management:** Efficiently manages application state using **React Hooks** for predictability and maintainability.
* **Rapid Styling with Tailwind CSS:** Utilizes a utility-first CSS framework for fast, consistent, and maintainable styling.
